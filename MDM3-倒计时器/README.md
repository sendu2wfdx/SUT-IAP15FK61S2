# MDM3-倒计时器

MDM3实验板的倒计时器程序，支持32s内的精确倒计时，默认计时时间为5s，时间显示精度10ms。

数码管显示及倒计时计数使用定时器中断实现，定时器1定时时间1ms。可以在倒计时过程中增减计时时间，暂停恢复计时等。倒计时结束，数码管归0，蜂鸣器鸣响三次后恢复默认状态。

##### 按键对应功能：

|     计时+10s      | **计时+1s** | **计时+0.1s** | **计时+10ms** |
| :---------------: | :---------: | :-----------: | :-----------: |
| **开始/结束计时** | **计时-1s** | **计时-0.1s** | **计时-10ms** |
